## 贡献指南

我们欢迎广大开发者贡献大家的智慧，让我们共同让它变得更完美.

### 贡献代码

1. Fork [openset/php2go](https://github.com/openset/php2go) 到本地.
2. 创建新的分支：

    ```shell
    $ git checkout -b new_feature
    ```

3. 编写代码。
4. Push 到你的分支:

    ```shell
    $ git push origin new_feature
    ```

5. 创建 Pull Request 并描述你完成的功能或者做出的修改。
